# 트랙: 소설 / 에세이 쓰기

이 파일은 claude-study 스킬의 글쓰기 트랙 가이드다.

## 1단계: 정보 수집

AskUserQuestion으로 한 번에 물어본다:
- 장르: 소설 / 에세이 / 블로그 글 / 기타
- 주제나 첫 아이디어 (없으면 Claude가 제안)
- 분량: 짧게(500자) / 보통(1500자) / 길게(3000자+)
- 톤: 가볍고 유머러스 / 진지하고 감성적 / 정보 전달형

## 2단계: 폴더 생성

    mkdir -p ~/projects/my-writing

## 3단계: 글쓰기

수집한 정보로 초안을 작성한다.

글쓰기 원칙:
- 첫 문장이 독자를 끌어당겨야 한다
- "~할 수 있습니다", "~에 대해 알아보겠습니다" 같은 AI 냄새 나는 표현 금지
- 문장은 짧게. 단락은 3~4줄 이내
- 구체적인 묘사와 감각적 표현 사용

장르별 특성:
- 소설: 장면 중심, 대화체 활용, 인물 감정 구체화
- 에세이: 개인 경험에서 출발해 보편적 통찰로 확장
- 블로그: 검색자 의도 고려, 소제목으로 구조화

## 4단계: 저장

초안 완성 후:

    # 파일명은 주제 기반으로 자동 생성
    # 예: ~/projects/my-writing/외로운-로봇-이야기.md

Write 도구로 저장한다.

## 5단계: 피드백 루프

"초안이 완성됐어요! 읽어보시고 수정하고 싶은 부분을 알려주세요.
예: '2번째 단락을 더 감성적으로', '결말을 바꿔줘', '전체적으로 더 짧게'"

수정 요청을 받으면 해당 부분만 수정해서 다시 보여준다.
