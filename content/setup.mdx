# 설치하기

쿠폰으로 7일 무료 체험을 시작했다면, 두 가지 방법 중 하나로 설치하면 됩니다.

---

## 방법 A — 데스크탑 앱 (추천)

터미널 없이 바로 사용할 수 있습니다.

<a href="https://claude.com/download" target="_blank" rel="noopener noreferrer">**claude.com/download**</a>에서 Windows용을 내려받아 설치하세요.

설치 후 앱을 열면 로그인 화면이 나타납니다. Claude 계정으로 로그인하면 바로 시작됩니다.

상단 탭에서 **코드**를 누르면 Claude Code를 바로 사용할 수 있습니다.

---

## 방법 B — 터미널 설치

터미널이 편하다면 아래 한 줄로 설치할 수 있습니다.

<StepCard step="1" title="설치 (PowerShell)">

시작 메뉴에서 **PowerShell**을 검색해 열고 아래를 실행하세요.

```powershell
irm https://claude.ai/install.ps1 | iex
```

<Callout type="tip">
`PS C:\`로 시작하는 프롬프트가 보이면 PowerShell이 맞습니다.  
CMD(명령 프롬프트)에서는 동작하지 않으니 주의하세요.
</Callout>

</StepCard>

<StepCard step="2" title="실행 & 로그인">

```powershell
claude
```

브라우저가 열리면서 로그인 화면이 나타납니다. Claude 계정으로 로그인하면 완료입니다.

</StepCard>

---

## 잘 안 된다면

**로그인 화면이 안 열림** → 아래를 직접 실행해 보세요.
```powershell
claude --login
```

문제가 해결되지 않으면 <a href="https://code.claude.com/docs/en/troubleshooting" target="_blank" rel="noopener noreferrer">공식 트러블슈팅 가이드</a>를 참고하세요.

---

## 스킬 설치하기

Claude Code가 실행되면 스킬 패키지를 설치해봅시다.

```powershell
curl -fsSL https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/install.sh | bash
```

설치 후 Claude Code 안에서 `/claude-study`를 입력하면 단계별 실습이 시작됩니다.
