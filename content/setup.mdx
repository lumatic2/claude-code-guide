# 설치하기

쿠폰으로 7일 무료 체험을 시작했다면, 아래 순서대로 따라오세요.

---

**1. 앱 설치**

<a href="https://claude.com/download" target="_blank" rel="noopener noreferrer">claude.com/download</a>에서 Windows용을 내려받아 설치하세요.

**2. 로그인**

앱을 열면 로그인 화면이 나타납니다. 쿠폰으로 가입한 Claude 계정으로 로그인하세요.

**3. Claude Code 시작**

상단 탭에서 **코드**를 누르고, **폴더 선택** 버튼으로 작업할 폴더를 지정하면 준비 완료입니다.

---

## 스킬 설치하기

Claude Code 입력창에 아래 내용을 **그대로 복사해서 붙여넣으세요.** Claude가 알아서 설치해줍니다.

```
아래 URL의 파일을 해당 경로에 저장해줘. 폴더가 없으면 만들어줘.

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/SKILL.md
→ ~/.claude/skills/claude-study/SKILL.md

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/references/track-website.md
→ ~/.claude/skills/claude-study/references/track-website.md

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/references/track-writing.md
→ ~/.claude/skills/claude-study/references/track-writing.md

완료되면 알려줘.
```

설치가 끝나면 `/claude-study`를 입력해 실습을 시작하세요.
