# 설치 & 첫 실행

Claude Code를 처음 시작하려면 몇 가지 준비가 필요합니다. 각 단계를 차근차근 따라가면 10분 안에 시작할 수 있습니다.

## 사전 요구사항

### Node.js 설치

**Node.js가 뭐예요?**  
프로그램을 실행하는 환경입니다. 마치 음악을 틀려면 스피커가 필요한 것처럼, Claude Code를 실행하려면 Node.js가 필요합니다.

**현재 설치되어 있는지 확인:**

```bash
node --version
```

결과가 `v18` 이상이면 준비 완료입니다. 예: `v20.11.0`

**설치가 필요하면:**

1. [nodejs.org](https://nodejs.org) 방문
2. 큰 버튼(**LTS 권장** - Long Term Support)을 클릭해서 다운로드
3. 설치 프로그램 실행 후 지시대로 진행
4. 설치 후 터미널을 새로 열고 위의 버전 확인 명령어 다시 실행

<Callout type="tip">
macOS/Linux에서 권한 오류 발생 시:  
npm 설정으로 sudo 없이 사용하도록 변경할 수 있습니다. 자세한 방법은 아래 트러블슈팅 참조.
</Callout>

### API 키 발급

**API 키가 뭐예요?**  
Anthropic 서버와 안전하게 통신하기 위한 고유한 암호 같은 것입니다. 은행 계좌번호처럼 소중하게 다뤄야 합니다.

**발급 방법:**

1. [console.anthropic.com](https://console.anthropic.com) 접속
2. 회원가입 또는 로그인 (Google/GitHub로도 가능)
3. 좌측 메뉴에서 **API Keys** 클릭
4. **Create Key** 버튼 클릭
5. 생성된 키가 나타남 (한 번만 보임!) → 복사해서 안전한 곳에 저장

<Callout type="warn">
키는 절대 공유하거나 GitHub에 올리지 마세요.  
누군가 이 키를 빌려가면 당신 계정이 악용될 수 있습니다.
</Callout>

## 설치 및 시작

<StepCard step="1" title="Claude Code 설치">

터미널(명령어 창)을 열고 다음을 입력합니다.  
npm(프로그램 설치 도구)을 통해 Claude Code를 받습니다.

```bash
npm install -g @anthropic-ai/claude-code
```

설치 완료 확인:

```bash
claude --version
```

결과: `Claude Code v0.x.x` 형태

</StepCard>

<StepCard step="2" title="API 키 설정">

**Windows (PowerShell 또는 명령 프롬프트):**

```powershell
$env:ANTHROPIC_API_KEY = "sk-ant-여기에-발급받은-키-붙여넣기"
```

**macOS / Linux (Terminal):**

```bash
export ANTHROPIC_API_KEY="sk-ant-여기에-발급받은-키-붙여넣기"
```

키 설정 확인:

```bash
echo $ANTHROPIC_API_KEY
```

(발급받은 키 일부가 보이면 성공)

<Callout type="tip">
**매번 입력 없이 자동 설정:**  
이 명령어를 매번 실행하는 것은 번거롭습니다.  
홈 폴더의 `.bashrc` (Linux/macOS) 또는 PowerShell 프로필에 추가하면 터미널 시작할 때마다 자동으로 설정됩니다.  
(방법은 아래 트러블슈팅 참조)
</Callout>

</StepCard>

<StepCard step="3" title="첫 실행">

준비가 완료되었습니다! 이제 Claude와 대화해봅시다.

```bash
claude
```

또는 특정 디렉터리(폴더)에서 작업하려면:

```bash
cd my-project
claude
```

**실제 화면 (예시):**

```
$ claude
✓ API key loaded
Welcome to Claude Code!

You: 안녕하세요! Python으로 "Hello, World!"를 출력하는 코드를 만들어 주세요.

Claude: 물론입니다!

    print("Hello, World!")

실행 방법:
1. 위의 코드를 hello.py 파일로 저장
2. 터미널에서 python hello.py 실행

You: (다음 질문 입력...)
```

이제 자유롭게 질문하고 코드를 작성할 수 있습니다.

<Callout type="tip">
`exit`를 입력하거나 `Ctrl+C`를 누르면 Claude와의 대화를 끝낼 수 있습니다.
</Callout>

</StepCard>

## 트러블슈팅

### "API 키 오류" 또는 "인증 실패"

**원인:** 키 앞뒤에 공백이 있거나 잘못된 키 복사

**해결:**
1. 발급받은 키를 다시 확인 (공백 없이)
2. 위의 Step 2에서 키를 다시 입력
3. `echo $ANTHROPIC_API_KEY`로 정확히 저장되었는지 확인

### "node command not found" 또는 "Node.js를 찾을 수 없음"

**원인:** Node.js가 설치되지 않았거나 PATH가 설정되지 않음

**해결:**
1. nodejs.org에서 LTS 버전 다시 다운로드
2. 설치 중 "Add to PATH" 옵션 체크 확인
3. 설치 후 새 터미널 창 열기
4. `node --version` 재확인

### "npm permission denied" (macOS/Linux)

**원인:** npm이 root 권한을 요구

**해결 (권장):**

```bash
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
export PATH=~/.npm-global/bin:$PATH
```

.bashrc 또는 .zshrc에 마지막 줄 추가하면 영구 설정됨.

### "npm install이 계속 실패"

**해결:**

```bash
npm cache clean --force
npm install -g @anthropic-ai/claude-code
```
