## 스킬 설치하기

Claude Code 입력창에 아래 내용을 **그대로 복사해서 붙여넣으세요.** Claude가 알아서 설치해줍니다.

```
아래 URL의 파일을 해당 경로에 저장해줘. 폴더가 없으면 만들어줘.

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/SKILL.md
→ ~/.claude/skills/claude-guide/SKILL.md

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/references/track-website.md
→ ~/.claude/skills/claude-guide/references/track-website.md

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/references/track-writing.md
→ ~/.claude/skills/claude-guide/references/track-writing.md

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/references/track-pdf.md
→ ~/.claude/skills/claude-guide/references/track-pdf.md

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/references/track-files.md
→ ~/.claude/skills/claude-guide/references/track-files.md

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/references/level-2-sandbox.md
→ ~/.claude/skills/claude-guide/references/level-2-sandbox.md

https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/skill/references/level-3-skill.md
→ ~/.claude/skills/claude-guide/references/level-3-skill.md

완료되면 알려줘.
```

설치가 끝나면 `/claude-guide`를 입력해 온보딩을 시작하세요.
