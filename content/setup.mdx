# 설치하기

Claude Pro 또는 Max 구독이 있다면 3단계로 시작할 수 있습니다.

<StepCard step="1" title="Node.js 확인">

터미널을 열고 아래를 실행하세요.

```bash
node --version
```

`v18` 이상이 나오면 다음 단계로 넘어가세요.

아무것도 안 나오면 → [nodejs.org](https://nodejs.org)에서 **LTS** 버튼을 눌러 설치 후 터미널을 새로 여세요.

</StepCard>

<StepCard step="2" title="Claude Code 설치">

```bash
npm install -g @anthropic-ai/claude-code
```

설치 확인:

```bash
claude --version
```

</StepCard>

<StepCard step="3" title="실행 & 로그인">

```bash
claude
```

브라우저가 열리면서 로그인 화면이 나타납니다.  
**Claude 구독 계정으로 로그인**하면 바로 시작됩니다.

<Callout type="tip">
처음 실행 시 권한 허용 화면이 몇 번 나옵니다. 모두 허용하고 진행하면 됩니다.
</Callout>

</StepCard>

---

## 잘 안 된다면

**`node: command not found`** → Node.js 미설치. [nodejs.org](https://nodejs.org)에서 LTS 설치 후 터미널 재시작.

**`npm permission denied`** (macOS/Linux) →
```bash
sudo npm install -g @anthropic-ai/claude-code
```

**로그인 화면이 안 열림** → 아래를 직접 실행해 보세요.
```bash
claude --login
```

---

## 스킬 설치하기

Claude Code가 실행되면 스킬 패키지를 설치해봅시다.

```bash
curl -fsSL https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/install.sh | bash
```

설치 후 Claude Code 안에서 `/claude-study`를 입력하면 단계별 실습이 시작됩니다.
