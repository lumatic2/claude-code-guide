# 설치하기

쿠폰으로 7일 무료 체험을 시작했다면, 아래 순서대로 따라오세요.

<StepCard step="1" title="앱 설치">

<a href="https://claude.com/download" target="_blank" rel="noopener noreferrer">**claude.com/download**</a>에서 Windows용을 내려받아 설치하세요.

</StepCard>

<StepCard step="2" title="로그인">

앱을 열면 로그인 화면이 나타납니다. 쿠폰으로 가입한 Claude 계정으로 로그인하세요.

</StepCard>

<StepCard step="3" title="Claude Code 시작">

상단 탭에서 **코드**를 누르면 Claude Code가 시작됩니다.

**폴더 선택** 버튼을 눌러 작업할 폴더를 지정하면 준비 완료입니다.

</StepCard>

---

## 스킬 설치하기

앱 하단 입력창에 아래 명령어를 붙여넣으세요.

```
/claude-study
```

<Callout type="tip">
스킬이 없다고 나오면 먼저 터미널에서 아래를 실행하세요.  
(앱 하단 입력창에서 `!` 접두어로 실행할 수 있습니다)

```
curl -fsSL https://raw.githubusercontent.com/lumatic2/claude-code-guide/master/install.sh | bash
```
</Callout>
