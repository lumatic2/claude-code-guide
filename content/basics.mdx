# 기본 사용법

## 슬래시 명령어

입력창에 `/`를 입력하면 사용 가능한 명령어 목록이 나타납니다. 확인된 명령어:

| 명령어 | 설명 |
|--------|------|
| `/compact` | 대화가 길어졌을 때 압축 |
| `/cost` | 지금까지 사용한 비용 확인 |

<Callout type="tip">
`/` 입력 후 자동완성 목록에서 더 많은 명령어를 확인할 수 있습니다.
</Callout>

## 단축키

앱 버전에서 사용 가능한 단축키는 입력창에 `@`를 입력하면 파일 첨부, `!`를 입력하면 셸 명령 실행을 빠르게 할 수 있습니다. 그 외 단축키는 앱 환경마다 다를 수 있습니다.

## 좋은 요청 작성법

Claude Code는 지시가 명확할수록 더 좋은 결과를 냅니다.

### 구체적으로 말하기

| 모호한 요청 | 구체적인 요청 |
|------------|-------------|
| "웹사이트 만들어줘" | "내 소개 페이지 만들어줘. 이름, 직업, 연락처 포함. 미니멀한 디자인으로." |
| "글 써줘" | "교토 여행기 블로그 글 써줘. 3박 4일, 감성적인 톤, 800자 내외." |
| "정리해줘" | "Downloads 폴더에서 1년 넘은 파일들만 Archive 폴더로 옮겨줘." |

### 맥락 주기

결과물이 어디에 쓰일지 알려주면 더 적합한 결과가 나옵니다.

```
"인스타그램 피드용 카드뉴스 텍스트 써줘. 
 주제는 '번아웃 극복 5가지 방법', 2030 직장인 대상."
```

### 단계별로 나누기

한 번에 너무 많이 요청하면 결과가 엇나갈 수 있습니다.

```
# 좋지 않은 예
"웹사이트 만들고 SEO 최적화하고 배포까지 해줘"

# 좋은 예
1단계: "소개 페이지 HTML 먼저 만들어줘"
2단계: (확인 후) "SEO 메타태그 추가해줘"
3단계: (확인 후) "Vercel에 배포해줘"
```

<Callout type="tip">
막히면 "어떻게 하면 좋을까?"라고 물어보세요. Claude Code가 방향을 제안해줍니다.
</Callout>

## 따라해보기

### 실습 1: 나만의 소개 페이지

<StepCard step={1} title="Claude Code 실행">
Claude 앱을 열고 상단 탭에서 **코드**를 누릅니다.
</StepCard>

<StepCard step={2} title="요청하기">
아래를 그대로 입력하거나 내용을 바꿔서 사용하세요.

```
내 소개 페이지 HTML 파일 만들어줘.
이름: 홍길동
직업: 프리랜서 디자이너
소개 한 줄: "공간과 경험을 디자인합니다"
연락처: hong@gmail.com
깔끔한 흰색 배경, 가운데 정렬로 만들어줘.
```
</StepCard>

<StepCard step={3} title="결과 확인 후 수정">
바탕화면에 생성된 `index.html` 파일을 더블클릭해 브라우저에서 열어보세요. 마음에 안 드는 부분은 바로 수정 요청하세요.
</StepCard>

### 실습 2: 텍스트 파일 요약

<StepCard step={1} title="파일 준비">
요약하고 싶은 텍스트 파일 하나를 준비합니다 (회의록, 기사, 긴 문서 등).
</StepCard>

<StepCard step={2} title="@ 기호로 파일 전달">

```
@회의록.txt 이 내용 핵심만 5줄로 요약해줘.
액션 아이템도 별도로 정리해줘.
```
</StepCard>

<StepCard step={3} title="결과 저장">
```
방금 요약 결과를 summary.txt 파일로 저장해줘.
```
</StepCard>

### 실습 3: 블로그 초안 작성

```
여행 블로그 글 초안 써줘.
주제: 제주도 오름 등반 후기
느낌: 잔잔하고 사색적인 톤
분량: 600~800자
파일명: jeju-blog.md로 저장해줘
```

<Callout type="tip">
초안을 받은 뒤 "2번째 단락 더 감성적으로" 같이 부분 수정을 요청하면 금방 완성도가 높아집니다.
</Callout>
