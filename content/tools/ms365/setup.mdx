## 2. add-in 활성화

설치만으로는 사이드바가 자동으로 뜨지 않습니다. 첫 사용 시 한 번 활성화해야 합니다.

| 운영체제 | 메뉴 위치 |
|---|---|
| **Windows** | 상단 리본 → **홈(Home)** → **추가 기능(Add-ins)** → **Claude** |
| **Mac** | 상단 메뉴 → **도구(Tools)** → **추가 기능(Add-ins)** → **Claude** |
| **Web (브라우저)** | 상단 리본 → **홈** → **추가 기능** → **Claude** |

Claude를 클릭하면 화면 오른쪽에 사이드바가 열립니다.

<Callout type="tip">
**한 번 활성화하면 이후엔 자동으로 사이드바가 따라옵니다.** 단, Excel · Word · PowerPoint 각 앱마다 처음 한 번씩 활성화해야 합니다.
</Callout>

---

## 3. Claude 계정으로 로그인

사이드바가 열리면 Claude 로그인 화면이 보입니다.

1. **Sign in** 버튼 클릭
2. 브라우저가 열리면서 claude.com 로그인 페이지로 이동
3. 평소 쓰는 Claude 계정(유료 플랜)으로 로그인
4. **Authorize** 클릭 → Office 앱 사이드바로 자동 복귀

로그인이 끝나면 사이드바 안에 채팅창이 나타나고, Claude가 현재 열린 파일을 인식합니다.

---

## 4. 잘 됐는지 확인

Excel을 열고 사이드바에 다음을 입력해보세요:

```
지금 열려 있는 시트 구조를 한 줄로 요약해줘.
```

Claude가 시트 이름·열 헤더를 짚어가며 답하면 정상 설치된 겁니다.

## 안 될 때

| 증상 | 해결 |
|---|---|
| 추가 기능 메뉴에 Claude가 없음 | Office 앱 완전히 종료 후 재실행 / Office 버전이 위 표 요건 이상인지 확인 |
| "관리자 권한이 필요" 메시지 | 회사 계정인 경우 — IT 관리자에게 Microsoft 365 Admin Center에서 third-party add-in 허용 요청 |
| 로그인 후 사이드바가 비어 있음 | Claude 계정이 **유료 플랜**인지 확인. 무료는 차단됨 |
| 파일을 인식 못 함 | 사이드바 새로고침 아이콘 → 그래도 안 되면 파일을 한 번 저장(Ctrl+S) 후 재시도 |
