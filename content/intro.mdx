# Claude Code 입문 소개

> **코드를 몰라도 됩니다. 무엇을 만들고 싶은지만 말하세요.**

Claude Code는 대화로 컴퓨터를 움직이는 AI 에이전트입니다. 채팅으로 지시하면 파일을 직접 만들고, 수정하고, 실행까지 해줍니다. 개발자만의 도구가 아닙니다 — 글을 쓰는 사람, 디자인하는 사람, 아이디어를 실행하고 싶은 모든 사람을 위한 도구입니다.

## 챗봇과 에이전트의 차이

AI 챗봇(ChatGPT, Claude.ai 등)은 **조언을 해주는** 도구입니다. Claude Code는 **직접 실행하는** 도구입니다.

| | 일반 AI 챗봇 | Claude Code |
|---|---|---|
| 홈페이지 만들기 | "이렇게 만드세요" 설명 제공 | 파일을 직접 생성, 서버 실행 |
| 블로그 글쓰기 | 초안 텍스트 제시 | 파일로 저장, 발행까지 |
| 폴더 정리 | 방법 설명 | 실제로 파일 이동/정리 |
| 이미지 일괄 처리 | 코드 예시 제공 | 직접 스크립트 작성 후 실행 |

<Callout type="tip">
Claude Code는 "조언가"가 아니라 "실행가"입니다. 시키면 직접 합니다.
</Callout>

## 나는 어떻게 쓸 수 있을까?

### 작가 · 블로거

```
"교토 여행 사진들 보고 블로그 글 초안 써줘. 따뜻한 느낌으로."
"이 글 맞춤법 검사하고 어색한 문장 다듬어줘."
"내 블로그 글 10개를 주제별로 분류해서 목차 파일 만들어줘."
```

### 디자이너 · 크리에이터

```
"내 포트폴리오 웹사이트 만들어줘. 작업물 갤러리 형태로."
"/drawing 브랜드 로고 시안 4개 만들어줘."
"이 이미지들 전부 800×800으로 리사이즈해줘."
```

### 직장인 · 기획자

```
"이 엑셀 데이터 보고 월별 매출 그래프 만들어줘."
"회의록 정리해서 액션 아이템만 추출해줘."
"이 폴더 안에 파일들 날짜별로 자동 분류해줘."
```

### 개발자

```
"이 버그 원인 찾아서 고쳐줘."
"테스트 코드 작성해줘."
"리팩토링해줘 — 중복 코드 제거 위주로."
```

## 실제로 어떻게 동작하나요?

홈페이지를 만들고 싶다고 가정해봅시다.

```
당신: 내 소개 페이지 만들어줘. 이름은 김민지, 
      UX 디자이너고 연락처 이메일 넣어줘.

Claude Code:
  ◆ 파일 생성 중...
    index.html 생성 ✓
    style.css 생성 ✓
  ✓ 로컬 서버 시작 → http://localhost:3000

당신: 배경색을 베이지로 바꿔줘.

Claude Code: style.css 수정 완료했습니다.
```

명령 한 줄에 파일이 만들어지고, 수정도 말 한마디면 됩니다.

## 이 가이드의 학습 순서

1. **설치 가이드** — Claude 앱 설치 & Claude Code 시작 (10분)
2. **기본 사용법** — 슬래시 명령어, 좋은 요청 작성법, 첫 실습
3. **실전 워크플로우** — 블로그 작성 / 웹사이트 만들기 / 파일 정리
4. **고급 설정** — CLAUDE.md로 나만의 규칙 만들기, 권한 설정
5. **치트시트** — 자주 쓰는 명령어 한 장 요약

순서대로 따라가면 이 가이드를 다 읽는 시간 안에 실제로 무언가를 만들 수 있습니다.
