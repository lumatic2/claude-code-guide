# Claude Code 입문 소개

Claude Code는 터미널 기반으로 코드 읽기, 수정, 검증까지 빠르게 수행할 수 있는 개발 에이전트입니다.

<Callout type="info" title="이 문서에서 다루는 범위">
기초 설치, 기본 워크플로우, 자주 쓰는 명령어를 중심으로 안내합니다.
</Callout>

<StepCard step="STEP 1" title="작업 디렉터리 준비">
프로젝트 루트에서 실행하는 습관을 들이면 경로 실수를 크게 줄일 수 있습니다.
</StepCard>

<StepCard step="STEP 2" title="작업 목표 명확화">
요청 시 목표, 범위, 제약, 완료 기준을 짧게 정리하면 품질이 안정됩니다.
</StepCard>

<StepCard step="STEP 3" title="검증 우선">
수정 후에는 반드시 빌드/테스트를 실행해 결과를 확인하세요.
</StepCard>
