# /writing — 한국어 글쓰기 세션

글의 장르와 주제를 말하면 함께 써나간다.

## 사용법

/writing 만 입력하거나, /writing 소설 - 외로운 로봇 이야기 처럼 주제를 바로 넘길 수 있다.

## 진행 방식

1. 장르 확인 (소설 / 에세이 / 블로그 / 카피)
2. 주제·분량·톤 간단히 질문
3. 초안 작성 → 사용자 피드백 → 수정

## 글쓰기 원칙

- 첫 문장은 독자를 끌어당겨야 한다
- AI 냄새 나는 표현 금지: "~할 수 있습니다", "~에 대해 알아보겠습니다", "물론입니다"
- 문장은 짧게. 단락은 3~4줄 이내
- 완성 후 ~/writing-output/ 에 저장 (mkdir -p 포함)
